Dwim Perl for Linux

This is a plain perl distribution of perl 5.14.2 built on Ubuntu 11.10. It is a very experimental version use it with caution.

Highlights:

  • perl 5.14.2
  • DBD::SQLite
  • Moose
  • Dancer
  • Plack
  • Starman

Dwim Perl for Windows

Dwim Perl for Windows is a Strawberry Perl derivative for Windows. It contains everything you will need for your Perl development.

We are waiting for the official release of Strawberry Perl 5.14 to build our first version. Till then you should download the package called "Padre on Strawberry Perl" that was the previous name of this package.

Future highlights:

  • Strawberry Perl
  • Moose, the post modern Object System
  • Perl::Critic, to police yourself
  • Perl::Tidy, to keep your code nice
  • Net::Telnet, to access the Linux boxes
  • Test::* for testing anything, anywhere
  • SQLite, to hold your data
  • MySQL and PostgreSQL drivers
  • Dancer, the easiest way to write web applications in Perl
  • Mojolicious, real-time web framework making web development fun again
  • Catalyst, the MVC Web Framework
  • Padre, the Perl IDE
  • ...

Current status: Waiting for Godot Strawberry Perl.
